博客搭建过程中的问题
Apr 30, 2024

Hexo

–Hexo是基于Node.js的静态网站生成器，支持Markdown语法，支持用户安装各种插件，且部署服务器十分方便，是搭建个人博客的良好选择。

Node.js

–JavaScript的运行环境，适合构建网络应用程序。

Markdown

–轻量级标记语言。语法随用随看，轻松易记。

Git & GitHub

git

–Ta 是一个分布式版本控制系统，可追踪文件修改历史——能够恢复文件历史版本；分支管理——可以轻松创建.切换.合并和删除分支；支持远程仓库；快速高效；本地化——每个开发者有完整代码仓库副本，不依赖网络连接就可以对本地进行版本控制和操作；灵活性；庞大的开源社区。

github

–Ta 是一个基于git的代码托管和开发协作平台。交互方式是网站，而Git是命令行。

静态和动态网站

–静态：网页内容是提前编写好的静态文件；缺乏用户交互——不会根据用户产生个性化内容；加载速度快；成本低；适合个人博客或者产品展示。
–动态：网页内容是通过服务器端和数据库处理用户请求；用户交互丰富；可以实现复杂应用和数据操作；例如淘宝官网等。

URL(uniform resource location)

–统一资源定位符；组成部分：协议.域名.端口.路径.携带参数.哈希值。
例如 https://www.baidu.com/s?d=hello&rsv_spt=1#5
—协议部分：https:
“//“分隔
—域名部分：www.baidu.com，(www.baidu)为二级域名，(.com)为顶级域名
“:”分隔
—端口部分（有默认值，可省略）：https默认端口号443；http默认80；Ftp默认21
—路径：”/“表示根目录，一般情况形如”www.baidu/路径1/路径2“
—参数部分：”?”to”#”，又称搜索部分..查询部分，参数之间使用”&”分隔。
—往后部分又称为“锚点”，非必须部分。

DNS(domain name server)

–域名解析服务——可将网址换为计算机所能理解的”IP地址”

IP(internet protocol)

网络之间互连的协议。定位一个设备地址&封装报文与其他设备交流。
特殊IP：
“localhost” 对应IPv4地址”127.0.0.1”,在浏览器中输入可查看本地运行的网页服务。
